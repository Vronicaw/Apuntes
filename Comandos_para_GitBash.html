Comandos básicos para la terminal de Git Bash en Windows


Este es un archivo básico de HTML que creé con el fin de guardar los comandos para la terminal de Git Bash. Estos comandos son los que aprendí e iré aprendiendo a medida que sigo estudiando programación.

Comandos:

  1. pwd: muestra la carpeta donde estoy
  2. ls: muestra los archivos de la carpeta donde estoy
  3. cd: me permite ingresar a una carpeta
  4. cd ..: me permite salir de una carpeta
  5. atom .: me permite abrir el editor de código
  6. mkdir: me permite crear una carpeta
  7. clear: me permite limpiar la pantalla
  8. cat: me permite ver el contenido de un archivo
  9. rm: me permite borrar un archivo o carpeta
  10. cp 'archivo' 'carpeta': copiar un archivo
  11. mv 'carpeta origen/archivo' 'carpeta nueva': mover un archivo
  12. git status: me muestra el estado actual del repositorio (archivos)
  13. git add 'archivo': guarda el archivo para hacerle commit
  14. git commit 'archivo': guarda el archivo para subir al repositorio

Recuerda que para subir todo a GitHub, tienes que crear el repositorio directamente en GitHub y luego copias la línea de 'git remote' en la terminal y luego copias la línea de git push. Cuando quieras agregar cambios a tu repositorio o nuevos archivos, solo debes hacer el mismo procedimiento con git add/git commit y luego escribir en la terminal git push.
